# PetNest 🐾 -A Pet Adoption Platform

A modern pet adoption platform where users can explore pets, submit adoption requests, and manage pet listings easily.

## 🌐 Live Site

https://pet-nest-eight.vercel.app

## 🎯 Purpose

PetNest helps connect loving families with pets in need of a home through a clean and user-friendly adoption platform.

## ✨ Features

- 🔍 Search and filter pets easily
- 🐶 Add and manage pet listings
- 🔐 Google Authentication system
- 📋 Adoption request management
- 📱 Fully responsive for all devices
- ⚡ Dynamic data fetching with MongoDB
- 🎨 Modern UI with Tailwind CSS
- 👤 User dashboard functionality

## 🛠️ Technologies Used

### Frontend

- Next.js
